# Online Retail Sales Performance Analysis

## Project Overview
This project analyzes the business performance of online retail sales in the United States from 2019 to 2022.
## Tools
- Power BI
- Power Query

## Dataset
The dataset includes transaction-level data such as:
Order ID, Order Date, Customer, Product Category, Region, Sales, Profit, Quantity, Discount, and Profit Margin.

## Analysis Objectives
- Analyze sales performance by product category
- Evaluate regional sales performance
- Identify profitability trends
- Support business performance evaluation

## Files
- Power BI Dashboard (.pbix)
- Analysis Report
- Dashboard Visualization
